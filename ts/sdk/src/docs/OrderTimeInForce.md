# OrderTimeInForce

The time-in-force policy for the order. By default, all orders are GTT. UNSPECIFIED is set to default.  GTT: Good Til Time  IOC: Immediate Or Cancel  FOK: Fill Or Kill     

## Enum

* `Gtt` (value: `'GTT'`)

* `Ioc` (value: `'IOC'`)

* `Fok` (value: `'FOK'`)

* `Unspecified` (value: `'UNSPECIFIED'`)
